<h1>"Perceptron do Zero: Implementação de um Neurônio Artificial em Python"</h1>

Este projeto é o meu "ponto de virada" na transição da linguagem C para Python. O objetivo foi construir a lógica de um neurônio artificial (Perceptron) do zero, sem usar bibliotecas prontas de IA, para entender a matemática por trás do aprendizado de máquina.
